# P Experiments - PC Cleaner 🚀

Удобная, легкая и быстрая утилита для автоматической очистки компьютера от системного мусора, написанная на Python с использованием графического интерфейса **Tkinter**. 

Программа работает в многопоточном режиме (GUI не зависает во время сканирования) и позволяет как оценивать общий объем мусора, так и безопасно удалять его.

---

## 🛠️ Основные возможности

* **Анализ в реальном времени:** Программа автоматически подсчитывает размер мусора при старте.
* **Кнопка «Обновить»:** Позволяет пересчитать объемы папок в любой момент времени без перезапуска утилиты.
* **Раздельная очистка:** Вы можете очистить только кэш браузера или только временные файлы Windows.
* **Безопасность:** Программа удаляет только кэш, логи и временные файлы, не затрагивая важные системные данные. Если файл заблокирован запущенной системой или игрой, утилита просто пропустит его без вылетов.
* **WinAPI интеграция:** Точный и мгновенный подсчет размера Корзины напрямую через ядро Windows.

---

## 📂 Что именно очищает программа?

1. **Временные файлы Windows:** Папки `C:\Windows\Temp` и `AppData\Local\Temp`.
2. **Кэш Google Chrome:** Папки `Cache`, `Code Cache\js` и `GPUCache` текущего профиля пользователя.
3. **Корзина Windows:** Полная очистка системной корзины.

## 💾 Как скачать готовый EXE (для обычных пользователей)

Если вы не хотите устанавливать Python и собирать проект из исходного кода, вы можете скачать уже скомпилированную версию программы:

1. Перейдите в раздел **[Releases](https://github.com/pavekscb/clear/releases)** на этом репозитории (справа на главной странице).
2. Скачайте архив с последней версией программы.
3. Распакуйте архив в удобное место и запустите файл `main.exe`.

> ⚠️ **Примечание:** Для полной очистки системной папки `C:\Windows\Temp` рекомендуется запускать скомпилированный `main.exe` **от имени Администратора** (правой кнопкой мыши -> Запуск от имени администратора).

## 💻 Инструкция по сборке из исходного кода (для разработчиков)

Если вы хотите запустить проект напрямую или самостоятельно собрать `.exe` файл, следуйте этим шагам:

### 1. Требования
У вас должен быть установлен **Python 3.x**. Сторонние библиотеки для работы программы (кроме PyInstaller для сборки) не требуются — всё написано на чистом WinAPI и встроенных модулях.

### 2. Клонирование репозитория
```bash
git clone [https://github.com/pavekscb/clear.git](https://github.com/pavekscb/clear.git)
cd clear

```

### 3. Запуск скрипта

Убедитесь, что все иконки интерфейса (`icon_all.png`, `icon_chrome.png`, `icon_refresh.png`, `icon_temp.png`, `icon_trash.png`, `logo.png`) лежат в одной папке со скриптом `main.py`, и выполните:

```bash
python main.py

```

### 4. Компиляция в самостоятельный .exe файл

Для сборки проекта в автономное Windows-приложение используется утилита `pyinstaller`.

* Установите сборщик:
```bash
pip install pyinstaller

```


* Выполните команду сборки (флаги автоматически упакуют графику внутрь и скроют консольное окно):
```bash
pyinstaller --noconfirm --onedir --windowed --add-data "icon_temp.png;." --add-data "icon_chrome.png;." --add-data "icon_trash.png;." --add-data "icon_all.png;." --add-data "icon_refresh.png;." --add-data "logo.png;." --icon "app_icon.ico" main.py

```



После завершения процесса готовое приложение со всеми ресурсами будет находиться в папке `dist/main/`. Вы можете переименовать папку `main` в `PC_Cleaner`, заархивировать её в `.zip` и загрузить в релизы GitHub!
